## Usage

### Install via Zed Extensions

1. Open Zed.
2. `cmd+shift+p` and select _zed: extensions_
3. Select _Cobalt2_ and Install
4. Select your Catppuccin theme in the dropdown shown after hitting ( `cmd+k`, `cmd+t` )
